Introduction

What is FOSS

FOSS stands for Free and Open Source Software, the 4 main points for FOSS are as follows:

  • The freedom to run the program as you wish, for any purpose (freedom 0).
  • The freedom to study how the program works, and change it so it does your computing as you wish (freedom 1). Access to the source code is a precondition for this.
  • The freedom to redistribute copies so you can help others (freedom 2).
  • The freedom to distribute copies of your modified versions to others (freedom 3). By doing this you can give the whole community a chance to benefit from your changes. Access to the source code is a precondition for this.

You might have heard of Open Source before, if yes it includes that. If not well just remember free as in freedom. This is in contrast to proprietary software, where the software is under restrictive copyright licensing and the source code is usually hidden from the users.

What is Linux

Linux is a kernel (Base for Operating System), and it's everywhere, your phone (android), to the smart TV, your smart home with all those functions, the websites you're browsing, almost everything in this world (and outside this world: satellites, robots in mars, etc.) runs on Linux. And it's everywhere because it's reliable, secure and customizable.

So the one we're talking about here are the users of Linux Based Desktop Operating Systems.

Even then there are hundreds of OS. Which makes it hard for UC to support all of them, hence we want to gather together to help each other.
